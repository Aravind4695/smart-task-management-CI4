# Smart Task Management System (CodeIgniter 4)

## Project Overview

This is a **Task Management Web Application** developed using CodeIgniter 4.
It helps users manage tasks efficiently with features like task creation, updating, deletion, and filtering.

---

## Features

* User Authentication (Login System)
* Task CRUD Operations (Create, Read, Update, Delete)
* Search & Filter Tasks
* Pagination for task listing
* AJAX-based dynamic operations (no page reload)
* Admin panel for managing users and tasks

---

## 🛠️ Tech Stack

* Backend: PHP (CodeIgniter 4)
* Frontend: HTML, CSS, Bootstrap, jQuery
* Database: MySQL
* AJAX for seamless UI

---

## Project Structure

* `app/Controllers` → Handles request logic
* `app/Models` → Database operations
* `app/Views` → UI pages
* `public/` → Entry point (index.php)

---

## Installation Steps

1. Clone the repository:

   ```bash
   git clone https://github.com/Aravind4695/smart-task-management-CI4.git
   ```

2. Move to project folder:

   ```bash
   cd smart-task-management-CI4
   ```

3. Configure `.env` file:

   * Set database name, username, password

4. Run project:

   ```bash
   php spark serve
   ```

5. Open browser:

   ```
   http://localhost:8080
   ```

##  Setup Instructions

After cloning the project, run:

```bash
composer install
```

This will install the required dependencies including the system folder.

Then configure `.env` file and run:

```bash
php spark serve
```
